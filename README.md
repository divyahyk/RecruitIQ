# 🧠 RecruitIQ — Intelligent Recruitment Platform

Zero-cost AI-powered recruitment platform built for modern talent teams.

## Features
- 👥 Profile Database — 5L+ candidates
- 🎯 AI JD Matching — 100-point scoring
- 📊 Kanban Tracker — 10-stage pipeline  
- 🗓️ Interview Scheduler — AI question bank
- 📱 Social Media — Auto-generated posts & banners
- 🤖 AI Sourcing — Boolean search & outreach

## Setup Secrets
Add in Space Settings → Repository Secrets:
- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`  
- `GROQ_API_KEY`
- `APP_PASSWORD`
